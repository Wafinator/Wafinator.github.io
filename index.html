Here are some of my Projects:

Realtime Chat Web Application

Realtime Chat is a web application developed using node.js in the backend and React.js in the frontend. This application uses ChatEngine.io and allows users to chat with each other in real-time. Users can send and receive messages/images from each other. Currently supports up to 1000 different users!




Sky Palette

Sky Palette is a weather application I made with HTML, CSS and JavaScript. It asks the user for a location and then uses API data from OpenWeatherMap to display basic weather information like current temperature, description, humidity, and wind speed. It has a simplistic UI to create a smooth experience for everyone.




NoteSwift

NoteSwift is a note taking web application I also made with HTML, CSS and JavaScript. Users are able to create, delete and store notes on this application. Created with a simple and easy to understand UI.
